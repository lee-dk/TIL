# Branch

## 1. Branch basic command

> - 브랜치 생성
>
> ```shell
> (master) $ git branch [branch name]
> ```
>
> - 브랜치 이동
>
> ```shell
> (master) $ git checkout [branch name]
> ```
> - 브랜치 생성 및 이동
> ```shell
> (master) $ git checkout -b [branch name]
> ```
>
> - 브랜치 목록
>
> ```shell
> (master) $ git branch
> ```
>
> - 브랜치 삭제
>
> ```shell
> (master) $ git branch -d [branch name]
> ```
